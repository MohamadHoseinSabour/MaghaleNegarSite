# MaghaleNegar Landing Page

صفحه معرفی و فروش افزونه وردپرس مقاله‌نگار

## Deploy

این پروژه روی GitHub Pages اجرا می‌شود. برای فعال‌سازی:

1. به **Settings** ریپو بروید
2. بخش **Pages** را باز کنید
3. **Source** را روی `main` branch و پوشه `/ (root)` تنظیم کنید
4. ذخیره کنید — سایت در چند دقیقه آماده می‌شود

## ساختار فایل‌ها

```
/
├── index.html      — صفحه اصلی لندینگ پیج
├── purchase.html   — صفحه خرید (فعلاً خالی)
├── style.css       — استایل‌های سایت
├── script.js       — جاوااسکریپت
└── README.md
```
